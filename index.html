# Cursor规则文件 - C语言学习网站代码展示效果实现指南 ## 项目概述 这是一个现代化的C语言学习网站，特色在于其精美的代码展示效果，包括终端风格的代码块设计和动态浮动元素。 ## 核心展示效果实现 ### 1. 终端风格代码块设计 #### 主体样式 ```css pre { background: #1a202c; /* 深色主背景 */ color: #e2e8f0; /* 浅色文字 */ padding: 1.5rem; /* 内边距 */ border-radius: 12px; /* 圆角 */ overflow-x: auto; /* 横向滚动 */ margin-bottom: 2rem; /* 底部间距 */ font-size: 0.9rem; /* 字体大小 */ line-height: 1.6; /* 行高 */ box-shadow: 0 4px 20px rgba(0, 0, 0, 0.4); /* 阴影效果 */ border: 1px solid #2d3748; /* 边框 */ position: relative; /* 相对定位，为伪元素准备 */ } ``` #### 顶部彩色装饰条 ```css pre::before { content: ''; /* 空内容 */ position: absolute; /* 绝对定位 */ top: 0; /* 顶部对齐 */ left: 0; /* 左侧对齐 */ right: 0; /* 右侧对齐 */ height: 30px; /* 装饰条高度 */ background: linear-gradient(90deg, /* 水平渐变 */ #fc8181 0%, /* 红色起始 */ #f6ad55 33%, /* 橙色 */ #68d391 66%, /* 绿色 */ #4fd1c7 100% /* 青色结束 */ ); border-radius: 12px 12px 0 0; /* 只有顶部圆角 */ } ``` #### macOS风格圆点装饰 ```css pre::after { content: '⬤ ⬤ ⬤'; /* 三个圆点 */ position: absolute; /* 绝对定位 */ top: 8px; /* 距离顶部8px */ left: 12px; /* 距离左侧12px */ color: #2d3748; /* 深色文字 */ font-size: 10px; /* 字体大小 */ letter-spacing: 4px; /* 字符间距 */ z-index: 2; /* 层级确保在装饰条之上 */ } ``` #### 代码内容区域 ```css pre code { margin-top: 25px; /* 为装饰条留出空间 */ display: block; /* 块级显示 */ position: relative; /* 相对定位 */ z-index: 1; /* 层级设置 */ } ``` ### 2. 语法高亮与特殊样式 #### 注释样式 ```css .code-comment { color: #6ee7b7; /* 绿色注释 */ font-style: italic; /* 斜体 */ } ``` #### 行内代码样式 ```css code { font-family: 'Courier New', Courier, monospace; /* 等宽字体 */ background: linear-gradient(135deg, rgba(229, 231, 235, 0.8) 0%, rgba(209, 213, 219, 0.8) 100% ); /* 渐变背景 */ color: #be123c; /* 红色文字 */ padding: 0.25rem 0.5rem; /* 内边距 */ border-radius: 6px; /* 圆角 */ font-size: 0.85em; /* 相对字体大小 */ border: 1px solid rgba(190, 18, 60, 0.2); /* 边框 */ backdrop-filter: blur(5px); /* 背景模糊效果 */ } ``` ### 3. 浮动动画效果实现 #### 浮动容器 ```css .floating-elements { position: absolute; /* 绝对定位 */ top: 0; /* 覆盖整个容器 */ left: 0; right: 0; bottom: 0; pointer-events: none; /* 不响应鼠标事件 */ } ``` #### 浮动元素样式 ```css .floating-element { position: absolute; /* 绝对定位 */ color: rgba(59, 130, 246, 0.3); /* 半透明蓝色 */ font-size: 2rem; /* 大字体 */ font-weight: bold; /* 粗体 */ animation: float 6s infinite ease-in-out; /* 浮动动画 */ } ``` #### 浮动动画关键帧 ```css @keyframes float { 0%, 100% { transform: translateY(0px) rotate(0deg); /* 起始和结束位置 */ opacity: 0.3; /* 透明度 */ } 25% { transform: translateY(-20px) rotate(5deg); /* 上浮并旋转 */ opacity: 0.6; /* 增加不透明度 */ } 50% { transform: translateY(-30px) rotate(-3deg); /* 最高点 */ opacity: 0.4; /* 调整透明度 */ } 75% { transform: translateY(-10px) rotate(2deg); /* 下降过程 */ opacity: 0.5; } } ``` #### 元素定位和延迟 ```css .floating-element:nth-child(1) { top: 20%; /* 垂直位置 */ left: 10%; /* 水平位置 */ animation-delay: 0s; /* 无延迟 */ } .floating-element:nth-child(2) { top: 60%; right: 15%; animation-delay: 1.5s; /* 1.5秒延迟 */ } .floating-element:nth-child(3) { bottom: 30%; left: 20%; animation-delay: 3s; /* 3秒延迟 */ } .floating-element:nth-child(4) { top: 40%; right: 40%; animation-delay: 4.5s; /* 4.5秒延迟 */ } ``` ### 4. 打字机动画效果 #### 代码行动画 ```css .code-line { color: #e5e7eb; /* 浅色文字 */ line-height: 1.8; /* 行高 */ margin: 0.5rem 0; /* 上下间距 */ opacity: 0; /* 初始透明 */ animation: typeWriter 3s infinite; /* 打字机动画 */ margin-top: 2rem; /* 顶部边距 */ } ``` #### 打字机动画关键帧 ```css @keyframes typeWriter { 0%, 20% { opacity: 0; transform: translateX(-10px); /* 从左侧滑入 */ } 25%, 75% { opacity: 1; transform: translateX(0); /* 正常位置 */ } 80%, 100% { opacity: 0; transform: translateX(10px); /* 向右侧滑出 */ } } ``` #### 动画延迟序列 ```css .code-line:nth-child(1) { animation-delay: 0.5s; } .code-line:nth-child(2) { animation-delay: 1s; } .code-line:nth-child(3) { animation-delay: 1.5s; } .code-line:nth-child(4) { animation-delay: 2s; } .code-line:nth-child(5) { animation-delay: 2.5s; } ``` ### 5. 玻璃态效果 #### 主容器玻璃态 ```css .hero-visual { background: linear-gradient(135deg, rgba(15, 23, 42, 0.95) 0%, rgba(30, 41, 59, 0.95) 100% ); /* 半透明渐变背景 */ backdrop-filter: blur(20px); /* 背景模糊 */ border-radius: 20px; /* 圆角 */ overflow: hidden; /* 隐藏溢出 */ box-shadow: 0 25px 50px rgba(0, 0, 0, 0.2); /* 深度阴影 */ } ``` ## 实现要点 ### 技术栈 - **CSS3**: 动画、渐变、模糊效果 - **伪元素**: ::before, ::after 用于装饰 - **CSS动画**: @keyframes 关键帧动画 - **定位**: absolute/relative 精确控制 ### 性能优化 - 使用 `transform` 而非 `left/top` 进行动画 - `will-change` 属性提示浏览器优化 - 合理使用 `pointer-events: none` 避免事件冲突 ### 浏览器兼容性 - 现代浏览器全面支持 - 渐进增强设计，老浏览器降级显示 - 使用厂商前缀确保兼容性 ## 自定义建议 ### 颜色主题 可以通过CSS变量轻松切换主题： ```css :root { --code-bg: #1a202c; --code-text: #e2e8f0; --accent-gradient: linear-gradient(90deg, #fc8181 0%, #4fd1c7 100%); } ``` ### 动画控制 可以通过媒体查询控制动画： ```css @media (prefers-reduced-motion: reduce) { .floating-element, .code-line { animation: none; } } ``` 这种实现方式创造了一个既美观又实用的代码展示效果，提升了用户的学习体验。
